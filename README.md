# Customer Segmentation

## Project Overview
This project performs customer segmentation using data analysis and unsupervised learning techniques. The goal is to identify distinct customer groups based on purchase behavior, demographics, and engagement metrics.

## Objectives
- Explore customer data and clean it for modeling
- Engineer features relevant to customer behavior
- Apply clustering methods to segment customers
- Visualize and interpret the resulting segments

## Dataset
The project uses customer-related data, including fields such as:
- Customer ID
- Purchase frequency and recency
- Total spend
- Demographic information
- Product preferences

## Tools and Dependencies
- Python
- pandas
- NumPy
- scikit-learn
- matplotlib
- seaborn
- Jupyter Notebook

## Project Structure
- `README.md` - project summary and usage notes
- `customer_segmentation.ipynb` - main notebook with analysis and modeling


## Usage
1. Open the notebook in Jupyter.
2. Load the dataset.
3. Run the preprocessing and feature engineering cells.
4. Fit clustering models and analyze the segments.
5. Review visualizations and interpret the customer groups.

## Results
The output includes customer segments with descriptions of behavior patterns and recommended actions for each group.
